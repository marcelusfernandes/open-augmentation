# Sistema Pessoal de Augmentacao Cognitiva com Claude Code

Arquitetura de alto nivel para um sistema pessoal onde arquivos persistentes funcionam como memoria operacional.
A IA acorda do zero a cada sessao, carrega identidade, contexto atual e conhecimento historico sob demanda.
O fluxo prioriza capturar primeiro e organizar depois, usando `memory/` como verdade quente e `knowledge/` como base fria.
