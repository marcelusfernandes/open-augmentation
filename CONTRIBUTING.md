# Contribuindo

Obrigado pelo interesse no **Augmentation**! Este repositório é um template público —
sinta-se livre pra usar, adaptar e propor melhorias.

## Como propor mudanças

A branch `main` é protegida: **toda alteração entra por Pull Request** (push direto é
bloqueado, exceto para mantenedores).

1. **Fork** este repositório.
2. Crie uma branch a partir da `main`: `git checkout -b minha-melhoria`.
3. Faça suas mudanças com commits claros.
4. Abra um **Pull Request** descrevendo o quê e o porquê.
5. Um mantenedor revisa e faz o merge (é necessária **1 aprovação**; comentários da
   revisão precisam estar resolvidos).

## Diretrizes

- **Foco:** PRs pequenos e com um objetivo são revisados mais rápido.
- **Nada pessoal:** `memory/`, `config/`, `recordings/` e `knowledge/_private/` são
  gitignored por design — não os adicione ao PR.
- **Skills/comandos:** descreva como testou a mudança.
- **Docs:** atualize o `README.md`/`CLAUDE.md` quando o comportamento mudar.

## Dúvidas

Abra uma *issue* descrevendo o caso de uso ou o problema.
